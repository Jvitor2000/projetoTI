# Nome do Projeto
TTRAIO Web

Objetivo: Aplicação Web de um site de vendas de varejo.
- Compra e venda de produtos
- Carrinho de Compras
- Exposição e Catálogo
- Pagamento
- Design 
- Avaliações
- Lista de Desejos (Wishlist)

Complemento (Se der tempo): ChatBot para conversa inicial, Tela de login diferente para vendedor, Rastreio de produto, Filtros Avançados, Design profissional, Cupons, Forma de pagamento variados, FAQ e Vídeos de Demonstração.

Linguagens: JavaScript - CSS - SQL - HTML - PHP
